# Kiin Bao Bao – Restaurant Website

A modern, responsive restaurant website built using HTML, CSS, and JavaScript.
This project demonstrates a clean user interface, interactive elements, and a multi-page structure inspired by real-world restaurant websites.

---

## Features

* Multi-page navigation (Home, Menu, 3-Course, Tasting)
* Interactive food cards with popup descriptions
* Dropdown menu navigation
* Smooth scroll to contact section
* Reusable vegan options section
* Custom background styling per page
* Responsive design for mobile and desktop

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
